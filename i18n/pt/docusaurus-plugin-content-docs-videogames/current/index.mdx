# 🎮 Videogames

Bem-vindo ao meu cantinho gamer! Aqui você encontrará minha coleção de jogos e perfis de plataforma em múltiplas plataformas.

:::tip Gamer de Coração
Além de tecnologia e desenvolvimento de software, os videogames são uma das minhas maiores paixões. Eu gosto de uma grande variedade de gêneros, de ação, aventura e RPGs a plataformas e jogos de corrida.
Eu jogo pela história, não pelos 100% de conquistas, a vida é muito curta para lograr tudo!
:::
