## 💾 Boot Duplo com Dois SSDs

Eu uso **dois SSDs separados**, cada um com sua própria instalação do Windows 11. Assim mantenho trabalho e jogos completamente isolados, com unidades separadas, ambientes separados e sem concessões.

## 🎮 Experiência Xbox em Tela Cheia (Modo Portátil)

No SSD de jogos eu inicio direto na **experiência Xbox em tela cheia**, a mesma interface estilo console, pensada para controle, que você encontra em portáteis como o ROG Ally. Ela transforma meu desktop em um console de tela grande: sem distrações da área de trabalho, sem precisar de mouse, é só ligar e jogar.

Eu a ativo com a **[Xbox Full Screen Experience Tool](https://github.com/8bit2qubit/XboxFullscreenExperienceTool)** do 8bit2qubit, um utilitário com interface gráfica de um clique que ativa com segurança a experiência oculta de jogos em tela cheia do Xbox no Windows 11.

:::info[Por que eu amo o modo portátil em um desktop]
- **Boot estilo console** direto para uma interface pensada para controle
- **Experiência portátil** para navegar por toda a minha biblioteca com o controle
- **Menos sobrecarga** do que a área de trabalho completa do Windows enquanto jogo
:::
