# 🎛️ Configuração de Acessórios para Xbox

Eu gosto de montar configurações imersivas para jogos de voo e corrida no Xbox e PC. Nesta página compartilho os acessórios que uso atualmente e os jogos que jogo com eles.

## ✈️ Configuração de Voo

- **Thrustmaster T-Flight Hotas One** para uma configuração clássica de joystick + manete de potência
- **Logitech G PRO Flight Yoke System com Rudder Pedals** para uma experiência de aviação civil mais realista

## 🏎️ Configuração de Corrida

- **Logitech G920** com volante + pedais
- **Logitech Driving Force Shifter**
- Montado em um **GT Omega Apex Racing Wheel Stand** para mais estabilidade e conforto
