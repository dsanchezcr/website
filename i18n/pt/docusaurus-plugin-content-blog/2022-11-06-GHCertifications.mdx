# Aprovadas nas certificações de Partners do GitHub

Estou muito feliz em compartilhar que acabei de ganhar as três Certificações de Parceiros do GitHub.
<!--truncate-->
As Certificações de Partners do GitHub são um conjunto de três exames que cobrem a plataforma GitHub e seu ecossistema. Os exames são projetados para testar seu conhecimento do GitHub.

Os três exames são:

*   [Ações do GitHub](https://credly.com/badges/e8a1a5a0-f95c-4bfd-b6cb-81ea1231153f)
*   [Administrador do GitHub](https://credly.com/badges/79b12418-8217-4c7b-a1a0-ad6f6cf407df)
*   [Segurança Avançada do GitHub](https://credly.com/badges/b32ce8d3-d9e1-423c-8d1a-958901f3d798)

[![GitHub Partner Certifications](pathname:///img/GH-Certs.png)](https://learn.microsoft.com/training/github)

[Coleções do Microsoft Learn](https://learn.microsoft.com/training/github) me ajudou muito a me preparar para os exames. Eu recomendo que você os verifique.

Estou ansioso para continuar aprendendo e compartilhando meu conhecimento com a comunidade.
