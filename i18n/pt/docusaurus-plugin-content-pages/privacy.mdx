# Privacidade e Transparência de Dados

Sua privacidade é importante. Esta página explica quais dados são coletados quando você visita este site, quais serviços de terceiros são usados e como suas informações são tratadas. Este site é um projeto pessoal — não um negócio — e está comprometido com a transparência e o respeito aos visitantes.

## Análises

Este site usa o **Google Analytics 4** (propriedade `G-18J431S7WG`) para entender como os visitantes usam o site. As seguintes configurações estão ativas:

- **A anonimização de IP** está habilitada (`anonymizeIP: true`) — seu endereço IP completo nunca é armazenado
- Os dados de análise são usados apenas para entender padrões de tráfego (visualizações de páginas, conteúdo popular, geografia dos visitantes em nível de país)
- Nenhuma informação de identificação pessoal (PII) é coletada através das análises
- O Google Analytics pode definir cookies (por exemplo, `_ga`, `_ga_*`) para distinguir visitantes únicos

Você pode optar por não participar do Google Analytics usando uma [extensão de navegador](https://tools.google.com/dlpage/gaoptout) ou bloqueando cookies de `googletagmanager.com`.

## Proteção contra Spam

A página de [Contato](/pt/contact) usa o **Google reCAPTCHA v3** para prevenir spam e abuso. O reCAPTCHA pode coletar informações de hardware e software (como dados do dispositivo e do aplicativo) e enviá-las ao Google para análise. Esses dados estão sujeitos à [Política de Privacidade](https://policies.google.com/privacy) e aos [Termos de Serviço](https://policies.google.com/terms) do Google.

O reCAPTCHA é executado apenas nas páginas onde é necessário (formulário de contato, assinatura do boletim) e não rastreia você pelo site.

## Serviços de Terceiros

Este site se integra com os seguintes serviços de terceiros:

| Serviço | Propósito | Dados Compartilhados |
|---------|-----------|----------------------|
| **Google Analytics** | Análise de tráfego | Visualizações anônimas, tipo de dispositivo, país |
| **Google reCAPTCHA v3** | Proteção contra spam | Impressão digital do dispositivo/navegador (gerenciado pelo Google) |
| **Giscus** | Comentários do blog (via GitHub) | Seu nome de usuário do GitHub (se comentar) |
| **Algolia** | Busca no site | Consultas de busca (anônimas) |
| **Open-Meteo** | Widget do clima | Sua localização aproximada (geolocalização do navegador, apenas se permitir) |
| **Azure Communication Services** | Envio de e-mails | Endereço de e-mail (formulário de contato, boletim) |
| **Azure Application Insights** | Monitoramento de API | Telemetria do servidor apenas (sem PII do usuário) |

## Formulário de Contato

Quando você envia o formulário de [Contato](/pt/contact):

1. Seu nome, e-mail e mensagem são validados no servidor
2. Um e-mail de verificação é enviado para confirmar seu endereço de e-mail
3. Após a verificação, a mensagem é encaminhada ao proprietário do site e uma confirmação é enviada a você
4. **Nenhum dado é armazenado permanentemente** — os tokens de verificação expiram após 24 horas
5. Limites de taxa são aplicados para prevenir abuso (3 envios por IP por hora)

## Boletim

Quando você se inscreve no boletim:

1. Seu **endereço de e-mail**, **preferência de idioma** e **frequência escolhida** (semanal ou mensal) são armazenados em um banco de dados seguro
2. Um e-mail de verificação confirma sua inscrição (dupla confirmação)
3. Você pode **cancelar sua inscrição a qualquer momento** através do link de um clique em cada e-mail do boletim
4. Você pode gerenciar suas preferências na página do [Boletim](/pt/newsletter)
5. Seus dados nunca são compartilhados com terceiros ou usados para marketing além do boletim deste site

## Cookies

Este site usa uma quantidade mínima de cookies:

| Cookie | Fonte | Propósito | Duração |
|--------|-------|-----------|---------|
| `_ga` | Google Analytics | Distinguir visitantes únicos | 2 anos |
| `_ga_*` | Google Analytics | Armazenar estado da sessão | 2 anos |
| `_GRECAPTCHA` | Google reCAPTCHA | Proteção contra spam | Sessão |

Nenhum cookie de publicidade é definido. Nenhum cookie de rastreamento além das análises é usado.

## Sem Publicidade — Nunca

**Este site nunca exibirá anúncios.** Este é um projeto pessoal criado para compartilhar conhecimento, não para gerar receita. Você nunca verá:

- Anúncios em banner ou anúncios pop-up
- Conteúdo patrocinado ou posicionamentos pagos
- Links de marketing de afiliados disfarçados de conteúdo
- Rastreamento para fins de segmentação publicitária

Este compromisso é permanente e incondicional.
