# 💱 Tipo de Cambio de Colones

**Paquetes NuGet y npm para tipos de cambio de Costa Rica**

Se trata de un repositorio de demostración sobre cómo compilar un paquete NuGet y publicarlo en GitHub Packages y NuGet.org. Además, cómo compilar un paquete npm y publicarlo en GitHub Packages y npmjs.com.

Es un caso real que obtiene el tipo de cambio de la [Ministerio de Hacienda de Costa Rica API](https://api.hacienda.go.cr/indicadores/tc) recuperando los valores de Colones, Dólares y Euros.

- 💻 Código fuente: [Echa un vistazo al repositorio](https://github.com/dsanchezcr/ColonesExchangeRate)
