# Privacidad y Transparencia de Datos

Su privacidad es importante. Esta página explica qué datos se recopilan cuando visita este sitio web, qué servicios de terceros se utilizan y cómo se maneja su información. Este sitio web es un proyecto personal — no un negocio — y está comprometido con la transparencia y el respeto a los visitantes.

## Analítica

Este sitio web utiliza **Google Analytics 4** (propiedad `G-18J431S7WG`) para entender cómo los visitantes usan el sitio. Las siguientes configuraciones están activas:

- **La anonimización de IP** está habilitada (`anonymizeIP: true`) — su dirección IP completa nunca se almacena
- Los datos de analítica se usan únicamente para entender patrones de tráfico (visitas a páginas, contenido popular, geografía de visitantes a nivel de país)
- No se recopila información de identificación personal (PII) a través de la analítica
- Google Analytics puede establecer cookies (por ejemplo, `_ga`, `_ga_*`) para distinguir visitantes únicos

Puede optar por no participar en Google Analytics usando una [extensión de navegador](https://tools.google.com/dlpage/gaoptout) o bloqueando cookies de `googletagmanager.com`.

## Protección contra Spam

La página de [Contacto](/es/contact) usa **Google reCAPTCHA v3** para prevenir spam y abuso. reCAPTCHA puede recopilar información de hardware y software (como datos del dispositivo y la aplicación) y enviarla a Google para su análisis. Estos datos están sujetos a la [Política de Privacidad](https://policies.google.com/privacy) y los [Términos de Servicio](https://policies.google.com/terms) de Google.

reCAPTCHA se ejecuta solo donde es necesario para el formulario de contacto y no lo rastrea a través del sitio.

## Servicios de Terceros

Este sitio web se integra con los siguientes servicios de terceros:

| Servicio | Propósito | Datos Compartidos |
|----------|-----------|-------------------|
| **Google Analytics** | Analítica de tráfico | Visitas anónimas, tipo de dispositivo, país |
| **Google reCAPTCHA v3** | Protección contra spam | Huella del dispositivo/navegador (gestionado por Google) |
| **Giscus** | Comentarios del blog (vía GitHub) | Su nombre de usuario de GitHub (si comenta) |
| **Algolia** | Búsqueda del sitio | Consultas de búsqueda (anónimas) |
| **Open-Meteo** | Widget del clima | Su ubicación aproximada (geolocalización del navegador, solo si lo permite) |
| **Azure Communication Services** | Envío de correos | Dirección de correo (formulario de contacto, boletín) |
| **Azure Application Insights** | Monitoreo de API | Telemetría del servidor y registros operativos (pueden incluir datos limitados de solicitudes) |

## Formulario de Contacto

Cuando envía el formulario de [Contacto](/es/contact):

1. Su nombre, correo electrónico y mensaje se validan en el servidor
2. Se envía un correo de verificación para confirmar su dirección de correo
3. Tras la verificación, el mensaje se reenvía al propietario del sitio y se le envía una confirmación
4. **No se almacenan datos permanentemente** — los datos de verificación pueden almacenarse temporalmente en Azure Table Storage y expiran automáticamente después de 24 horas
5. Se aplica un límite de velocidad para prevenir abuso (3 envíos por IP por hora)

## Boletín

Cuando se suscribe al boletín:

1. Su **dirección de correo electrónico**, **preferencia de idioma** y **frecuencia elegida** (semanal o mensual) se almacenan en una base de datos segura
2. Un correo de verificación confirma su suscripción (doble confirmación)
3. Puede **cancelar su suscripción en cualquier momento** mediante el enlace de cancelación en cada correo del boletín
4. Puede gestionar sus preferencias en la página del [Boletín](/es/newsletter)
5. Sus datos nunca se comparten con terceros ni se usan para marketing más allá del boletín de este sitio web

## Cookies

Este sitio web utiliza una cantidad mínima de cookies:

| Cookie | Fuente | Propósito | Duración |
|--------|--------|-----------|----------|
| `_ga` | Google Analytics | Distinguir visitantes únicos | 2 años |
| `_ga_*` | Google Analytics | Almacenar estado de sesión | 2 años |
| `_GRECAPTCHA` | Google reCAPTCHA | Protección contra spam | Sesión |

No se establecen cookies de publicidad. No se utilizan cookies de rastreo más allá de la analítica.

## Sin Publicidad — Nunca

**Este sitio web nunca mostrará anuncios.** Este es un proyecto personal creado para compartir conocimiento, no para generar ingresos. Nunca verá:

- Anuncios de banner o anuncios emergentes
- Contenido patrocinado o colocaciones pagadas
- Enlaces de marketing de afiliados disfrazados de contenido
- Rastreo para fines de segmentación publicitaria

Este compromiso es permanente e incondicional.
