# Pasé las certificaciones de socios de GitHub

Estoy emocionado de compartir que acabo de obtener las tres certificaciones de socios de GitHub.
<!--truncate-->
Las certificaciones de socios de GitHub son un conjunto de tres exámenes que cubren la plataforma GitHub y su ecosistema. Los exámenes están diseñados para poner a prueba su conocimiento de GitHub.

Los tres exámenes son:

*   [Acciones de GitHub](https://credly.com/badges/e8a1a5a0-f95c-4bfd-b6cb-81ea1231153f)
*   [Administrador de GitHub](https://credly.com/badges/79b12418-8217-4c7b-a1a0-ad6f6cf407df)
*   [Seguridad Avanzada de GitHub](https://credly.com/badges/b32ce8d3-d9e1-423c-8d1a-958901f3d798)

[![GitHub Partner Certifications](pathname:///img/GH-Certs.png)](https://learn.microsoft.com/training/github)

Las [colecciones de Microsoft Learn](https://learn.microsoft.com/training/github) me ayudaron mucho a prepararme para los exámenes. Te recomiendo que los revises.

Tengo muchas ganas de seguir aprendiendo y compartiendo mis conocimientos con la comunidad.
