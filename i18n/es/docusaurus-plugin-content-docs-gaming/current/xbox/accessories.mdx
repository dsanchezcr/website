# 🎛️ Configuración de Accesorios para Xbox

Me gusta crear configuraciones inmersivas para juegos de vuelo y carreras en Xbox y PC. En esta página comparto los accesorios que uso actualmente y los juegos que disfruto con ellos.

## ✈️ Configuración de Vuelo

- **Thrustmaster T-Flight Hotas One** para una configuración clásica de joystick + acelerador
- **Logitech G PRO Flight Yoke System con Rudder Pedals** para una experiencia de aviación civil más realista

## 🏎️ Configuración de Carreras

- **Logitech G920** con volante + pedales
- **Logitech Driving Force Shifter**
- Montado en un **GT Omega Apex Racing Wheel Stand** para mayor estabilidad y comodidad

## 🎮 Juegos que Juego con Estos Accesorios

- **Microsoft Flight Simulator**
- **Ace Combat**
- **Chorus**
- **Forza Horizon**
- **SnowRunner**

También estoy explorando más juegos compatibles con estos accesorios, especialmente a través de Game Pass y nuevos lanzamientos.
